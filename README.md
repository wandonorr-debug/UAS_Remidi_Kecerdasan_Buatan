# Prediksi Kelayakan Keringanan UKT Mahasiswa (REMIDI UAS WANDONO)

Proyek ini bertujuan untuk menentukan kelayakan mahasiswa dalam menerima keringanan UKT menggunakan teknik Machine Learning. 

### Metode yang Digunakan:
1. **K-Nearest Neighbors (KNN)**
2. **Random Forest Classifier**
3. **Support Vector Machine (SVM)**

### Hasil Analisis:
Berdasarkan pengujian, model terbaik dipilih berdasarkan akurasi, precision, dan recall tertinggi untuk mengisi data pada `testing_prediksi_mhs.csv`.

### Cara Menjalankan:
1. Clone repositori ini.
2. Pastikan library `pandas`, `scikit-learn`, `matplotlib`, dan `seaborn` sudah terinstall.
3. Jalankan file `.ipynb` di Jupyter Notebook atau Google Colab.
